 Демонстрация возможностей языка CSS для форматирования и визуального оформления веб-страниц. Проект наглядно показывает принцип разделения содержимого (HTML) и его представления (CSS), а также знакомит с основными понятиями: каскадность, наследование, методы подключения стилей и различные свойства форматирования текста.
